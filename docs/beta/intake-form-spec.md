# 신청 폼 (구글폼) 항목 명세

PII 최소수집 원칙. PO 비공개 영역(이메일)은 시트·이슈·코멘트에 일절 기재 금지.

| # | 질문 | 타입 | 필수 | 비고 |
|---|---|---|---|---|
| 1 | 앱에서 쓸 표기(닉네임, 실명 X) | 단답 | ✅ | 시트에는 `tester_id`(T01~T10)로 치환해 기록 |
| 2 | iOS 기기 모델 / 버전 | 단답 | ✅ | 디바이스 분포 확인 |
| 3 | 주 평균 운동 횟수 | 단일 선택 (1 / 2 / 3+) | ✅ | P1 적합도 |
| 4 | 단/복식 비중 | 단일 선택 (복식 위주 / 단식 위주 / 반반) | ✅ | 복식 위주 우선 선발 |
| 5 | TestFlight 초대용 Apple ID 이메일 | 단답 | ✅ | **PO 전용 비공개 영역**. QA 시트로 옮기지 말 것 |
| 6 | 동의: 익명 분석 이벤트와 자유 피드백을 제품 개선에 활용함 | 체크박스 | ✅ | 동의 없으면 자동 탈락 |
| 7 | 추가로 남기고 싶은 말 | 자유 (선택) | ❌ | 시트 Qual 탭 입력 시 마스킹 |

## 스크리닝 우선순위

1. P1 적합 (주 1회 이상 + 복식 위주) → 우선 선발
2. 동의 + iOS 기기 정상
3. 5명 미달 시 PO가 추가 컨택, 10명 초과 시 대기 리스트로 보관

## PII 처리

- 이메일/실명: PO 1Password 또는 별도 비공개 노션 페이지 보관, 30일 후 폐기
- 닉네임: tester_id로 즉시 치환, 시트·이슈·코멘트에는 치환된 ID만 사용
