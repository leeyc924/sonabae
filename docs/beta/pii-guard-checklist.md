# 베타 운영 PII 가드 체크리스트

NAS-13 §5 원칙을 운영 산출물에 적용. 위반 발견 시 즉시 마스킹·삭제 후 CEO 보고.

## 매 산출물 (시트/리포트/이슈 코멘트) 발행 전
- [ ] 실명·전화·이메일 평문 0건
- [ ] 사용자 닉네임은 `tester_id`(T01~T10)로 치환됨
- [ ] 상대 별명·파트너 별명은 `[상대별명]` 자리표시자
- [ ] 메모/raw quote는 식별 가능 단어(직장명, 클럽명 등) 마스킹 후 인용

## 스크린샷 첨부 시
- [ ] 닉네임/메모/상대 영역 모자이크
- [ ] 디바이스 알림창 미노출 (다른 앱 알림 포함)
- [ ] 분석 이벤트 페이로드 캡처 시 별명 hash조차 노출 금지

## 외부 공유 / LLM 입력
- [ ] raw quote 그대로 송출 금지, 요약문만
- [ ] 베타 시트 링크는 회사 도메인 내부 공유만, 외부 공개 금지

## TestFlight 이메일

- [ ] QA 시트·이슈·코멘트에 일절 기재 금지
- [ ] 베타 종료 +30일 내 폐기
