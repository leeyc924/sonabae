# 소나배 베타 운영 (NAS-23)

C-Q4 베타 모집 + 피드백 수집의 운영 산출물 디렉터리.

| 파일 | 용도 |
|---|---|
| `recruit-message.md` | CH-A(회사 동호회 슬랙) / CH-B(PO 네트워크 DM) 모집 카피 |
| `intake-form-spec.md` | 신청 구글폼 항목 명세 (PII 최소수집) |
| `feedback-sheet-template.csv` | 운영 시트 4탭(Roster / Quant / Qual / Buglog) 컬럼 헤더 |
| `weekly-report-template.md` | 1주차/2주차 리포트 템플릿 |
| `pii-guard-checklist.md` | 시트·코멘트·리포트·스크린샷 PII 가드 체크리스트 |
| `bug-routing.md` | 결함/요청 라우팅 규칙 (bug → child issue, usability → PO) |

운영 케이던스·합격 기준·전제 의존성은 NAS-23 plan 문서를 단일 출처로 사용한다.
